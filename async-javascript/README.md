# Async JavaScript – Event Loop Practice

This module focuses on understanding how JavaScript handles asynchronous execution using the **Event Loop**.

The exercises demonstrate how the **Call Stack**, **Microtask Queue**, and **Macrotask Queue** interact when executing asynchronous code.

---

## Topics Covered

- Call Stack execution
- `setTimeout` and Macrotasks
- Promises and Microtasks
- Microtask vs Macrotask priority
- Advanced Event Loop scenarios

---

## Files

- `01-call-stack.js`
- `02-setTimeout.js`
- `03-promises.js`
- `04-microtask-vs-macrotask.js`
- `05-advanced-event-loop.js`

---

## Goal

Build a strong understanding of JavaScript execution order and asynchronous behavior, which is essential for working with APIs and modern frontend frameworks.
